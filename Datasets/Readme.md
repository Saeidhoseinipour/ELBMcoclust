# **Classic3** 
 dataset~\citep{dhillon2003information} containing 3891 documents by merging the popular MEDLINE, CISI, and CRANFIELD sets. MEDLINE consists of 1033 abstracts from “Medical” papers, CISI consists of 1460 abstracts from “Information Retrieval” papers, and CRANFIELD consists of 1398 abstracts from “Aeronautical Systems” papers.
